Featured Project

WarehouseBot — Python Robot Simulator

WarehouseBot is a Python-based warehouse delivery robot simulator built using Object-Oriented Programming. The robot can move step by step, perform pickup and drop tasks, track battery usage, and store movement history.


I used Matplotlib to visualize the robot path, pickup and drop positions, and battery level. This project is helping me understand programming logic, task handling, and how systems can be designed in a structured way.
